# InfinityLens369 v1.1.0 — Public Hardening

v1.1.0 is the first post-public-release hardening pass. It keeps the v1.0 visual engine intact and focuses on making the app easier to demo, easier to recover, and easier for new users to report issues.

## Highlights

- Added a WebGL2 support check and fallback message.
- Added **Safe Mode** for older devices, projectors, and lower-performance public demos.
- Added **Reset Visuals** to return to the default release scene.
- Expanded supported audio-file detection by MIME type and extension.
- Added clearer playback/decode notices when a browser cannot play a file.
- Added a visible stage notice overlay and control-panel system status card.
- Added `0` as a Safe Mode hotkey.
- Included browser support state inside exported JSON receipts.
- Added public GitHub issue templates for bug reports, visual ideas, and feature requests.

## Safe Mode behavior

Safe Mode switches to a calmer configuration:

- Cosmic Drift
- Abyss Cyan palette
- Audio reactive off
- Low motion pressure
- Low audio drive
- Reduced glow
- Overlays off

It is intended as a public demo safety rail, not as a separate visual claim.

## Keyboard shortcuts

```text
Space  play / pause
C      cinematic view
F      browser fullscreen
N      next trip preset
R      random trip
S      slow flow
A      auto trip
0      safe mode
1      Dream motion profile
2      Cruise motion profile
3      Live motion profile
4      Warp motion profile
```

## Claim boundary

InfinityLens369 remains claim-safe creative software: an open-source art, math, shader, and audio-reactive visualization tool. It is not a physics proof, consciousness proof, medical tool, or scientific evidence engine.

## Suggested demo flow

1. Open the app.
2. Drop a song.
3. Press **Play portal**.
4. Press **C** for Cinematic.
5. Press **F** for Fullscreen.
6. Use **N**, **R**, and **1–4** to perform.
7. Press **S** or **0** if the visual motion gets too intense.
8. Export a PNG frame or JSON receipt.
