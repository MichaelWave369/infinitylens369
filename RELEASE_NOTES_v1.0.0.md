# InfinityLens369 v1.0.0 Release Notes

**Release name:** Drop a song. Open a portal.

InfinityLens369 v1.0.0 is the first public studio release of the browser-based, local-first, audio-reactive visualization engine.

## What is included

- Vite + React + TypeScript app shell
- WebGL2 shader visual stage
- Local drag/drop audio playback
- Web Audio feature extraction for bass, mids, highs, energy, beat pulse, and waveform
- Eight visual modes:
  - Black Hole Lens
  - Cosmic Drift
  - Pixel Melt
  - Kaleido Trip
  - Tunnel Bloom
  - Acid Melt
  - Mandelbrot
  - Julia Mirror
- Trip preset cycle
- Random Trip generator
- Auto Trip mode
- Audio speed / drive control
- Motion profiles:
  - Dream
  - Cruise
  - Live
  - Warp
- Slow Flow safety/calming button
- Cinematic view
- Browser fullscreen control
- Keyboard performance shortcuts
- Phi spiral, 3-6-9 grid, and equation overlays
- Visual address save/copy
- Receipt JSON export
- PNG frame export
- GitHub Pages workflow

## Keyboard shortcuts

```text
Space  play / pause
C      cinematic view
F      browser fullscreen
N      next trip preset
R      random trip
S      slow flow
A      auto trip
1      Dream motion profile
2      Cruise motion profile
3      Live motion profile
4      Warp motion profile
```

## Claim boundary

InfinityLens369 is an art, math, and software visualization tool. It is not a physics proof, consciousness proof, medical tool, diagnostic tool, or empirical evidence engine.

## Suggested demo flow

1. Open the app.
2. Drop in an audio file.
3. Press **Play portal**.
4. Press **Cinematic** or **F** for fullscreen.
5. Press **R** for random trip.
6. Press **1-4** to control motion intensity.
7. Press **S** if the motion becomes too intense.
8. Export a PNG frame or JSON receipt.
