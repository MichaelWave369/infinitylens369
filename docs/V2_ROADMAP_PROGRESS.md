# InfinityLens369 v2.0 Roadmap Progress

This tracker keeps the v1.x sprint ladder visible as InfinityLens369 moves from v1.0 toward v2.0.

## Completed

- **v1.0.0** — first public release.
- **v1.1.0** — public hardening, Safe Mode, WebGL fallback, reset, issue templates.
- **v1.2.0** — Transition Engine with Bloom, Warp, Glitch, Fade, and Pulse bridges.
- **v1.3.0** — Audio Engine v2 with smoother response, per-band drives, and beat punch.

## In progress

- **v1.4.0** — Liquid Light Pack.

## v1.4 target

- Add soft/hypnotic Liquid Light presets.
- Keep existing shader modes stable.
- Favor beauty, slow flow, projection readiness, and smoother audio response.
- Promote the strongest Liquid Light preset into a standalone shader in a follow-up sprint.

## Next after v1.4

- **v1.5** — standalone Liquid Light shader winner or Machine Cathedral visual pack start.
- **v1.6** — Preset Studio / import-export preset packs.
- **v1.7** — Recording/export workflow.
- **v1.8** — Performance Controls v2.
- **v1.9** — Layer System foundation.
- **v2.0** — launch polish, gallery, tutorial/demo mode, public release package.
