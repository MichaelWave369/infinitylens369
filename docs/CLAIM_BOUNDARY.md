# InfinityLens369 Claim Boundary

InfinityLens369 is an open-source fractal and audio-reactive visualization engine.

It combines browser audio analysis, GPU rendering, fractal formulas, symbolic overlays, and replayable visual addresses.

## Accepted claim level

InfinityLens369 can safely claim to be:

- A browser-based visualizer
- A fractal exploration tool
- A local-first audio-reactive art engine
- A math-inspired graphics playground
- A visual-address and scene-receipt experiment
- A source generator for music videos, projection art, and posters

## Not claimed

InfinityLens369 does **not** claim to be:

- A proof of hidden physics
- A consciousness detector
- A medical or mental health tool
- A spiritual authority system
- A scientific instrument without validation
- Evidence that a generated visual represents real-world causality

## Visual address status

A visual address is a replay cue for a scene state.

It may record:

- Formula mode
- Camera center
- Zoom
- Rotation
- Palette
- Overlay choices
- Audio time
- Creation timestamp

It does not prove a physical, spiritual, or biological claim.

## Public communication rule

Use language like:

> InfinityLens369 creates music-reactive fractal art and replayable visual scene addresses.

Avoid language like:

> InfinityLens369 reveals the true structure of reality.

## Canon-safe summary

**Art / math / software:** yes.  
**Physics proof / medical proof / consciousness proof:** no.
