# InfinityLens369 Roadmap

## v0.1 — Fractal song portal

Goal: make the magic obvious in seconds.

- Local audio drop
- Play/pause local audio
- Web Audio analyzer
- Bass/mid/high/energy feature extraction
- WebGL2 Mandelbrot shader
- Julia mirror mode
- Audio-reactive zoom, rotation, glow, and color
- Phi spiral overlay
- 3-6-9 overlay
- Equation/signal overlay
- Visual address save/copy
- Receipt JSON export
- PNG frame export

## v0.2 — Atlas memory

Goal: make discoveries replayable.

- Bookmark gallery
- Import visual address
- Export/import scene packs
- LocalStorage scene library
- Named camera paths
- Safer deep-zoom boundaries
- Better mobile controls

## v0.3 — Music video loop engine

Goal: help artists generate source visuals quickly.

- WebM capture
- Beat-synced camera path presets
- Track section markers
- Loop templates
- Poster-frame capture
- High-resolution still export

## v0.4 — Formula pack

Goal: expand beyond Mandelbrot/Julia.

- Burning Ship
- Newton fractals
- Mandelbar / Tricorn
- Strange attractors
- Reaction-diffusion field
- Wave interference field

## v0.5 — Performance mode

Goal: make it useful live.

- Fullscreen controls
- Keyboard shortcuts
- MIDI/web controller mapping
- Preset hotkeys
- Reduced UI mode

## v1.0 — Fractal Atlas Studio

Goal: ship the first stable creative engine.

- Stable visual-address format
- Scene library
- Preset packs
- Video loop export
- Shader pack architecture
- Public gallery-ready examples
- Clean documentation and contribution guide
