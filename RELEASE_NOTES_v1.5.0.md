# InfinityLens369 v1.5.0 — Machine Cathedral Pack

Sprint 5 toward v2.0 adds the first Machine Cathedral family to the live app.

## What shipped

- Added **Machine Cathedral Pack** as a live one-click preset family.
- Added four cyber/geometry presets:
  - **Circuit Cathedral** — glowing circuit-temple geometry with grid discipline.
  - **Glyph Rain** — symbolic data rain with pixel-glitch energy.
  - **Neon Lattice** — wireframe tunnel lattice and node-field pulse.
  - **Vector Shrine** — retro vector altar with equations and mirror recursion.
- Added a Machine Cathedral control-panel section below Liquid Light.
- Added Machine Cathedral presets into Next Trip, Random Trip, and Auto Trip.
- Added warp-style transition behavior for Machine Cathedral preset changes.
- Bumped app/package version to `v1.5.0` / `1.5.0`.
- Updated README with the v1.5 feature set.

## Design stance

This sprint intentionally uses the stable shader routes already proven green:

- Tunnel Bloom
- Pixel Melt
- Julia Mirror

That keeps the public release moving while leaving room for future standalone Machine Cathedral shaders in v2.0.

## Suggested test flow

```text
1. npm run build
2. npm run dev
3. Drop a song
4. Click Circuit Cathedral, Glyph Rain, Neon Lattice, and Vector Shrine
5. Try Next Trip, Random Trip, and Auto Trip
6. Confirm Machine Cathedral presets use warp-style transitions
7. Export a JSON receipt from one Machine Cathedral scene
```

## Claim boundary

InfinityLens369 remains claim-safe creative software. These visuals are art/math/software experiences and replay cues, not physics, medical, spiritual, or consciousness evidence.
