# Contributing to InfinityLens369

Thanks for helping build the lens.

InfinityLens369 is meant to be playful, beautiful, technically grounded, and claim-safe.

## Good contributions

- New shader formulas
- Palette presets
- Audio analysis improvements
- UI/UX improvements
- Accessibility improvements
- Performance optimization
- Export tools
- Visual address import/export improvements
- Documentation and examples

## Claim-safe rule

Please keep public language grounded.

Good:

> This shader creates a music-reactive fractal field.

Avoid:

> This proves a hidden structure of reality.

## Local setup

```bash
npm install
npm run dev
```

## Pull request checklist

Before opening a PR, please check:

- The app builds with `npm run build`
- The UI still works with no audio loaded
- The app still keeps audio local in the browser
- New presets or shaders have clear names
- Any big claims are framed as art/math/software, not proof

## Project vibe

Build wonder. Keep receipts. Stay grounded.
